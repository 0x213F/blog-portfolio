Q-Learning Pong

ECE 448 - Artificial Intelligence
This web application is a simplified version of Atari's 1972 iconic video game "Pong." This is an unsupervised machine learning technique where the CPU learns how to play the game after many trials. Click below to play and use the ↑ and ↓ arrow keys to move.

Little Computer 3

Simulates a 16-bit CPU (LC3 ISA).
The goal of this project was to make a better assembly code learning environment for aspiring computer engineers. This web application brings the modern feel of an IDE without abstracting core computing concepts. An advanced debugger includes a program reverser, interactive memory dump, register contents, and much more.

HueDJ

Record light shows
Using the Phillips Hue API, this application provides a simple and scalable interface to make live light effects right from your home. Orchestrate the lights with your fingertips by popping, fading, and strobing the lights. Record your session to play it back later with your favorite song.
